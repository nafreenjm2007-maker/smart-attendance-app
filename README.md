🎓 Smart Attendance App
A web-based smart attendance management system built for schools and colleges.

🌐 Live Demo
👉 https://smart-attendance-app-gamma.vercel.app

👥 User Roles
Admin – Manage teachers, students, and view reports
Teacher – Mark attendance for their classes
Student – View their own attendance records

🛠️ Tech Stack
Frontend: React.js, deployed on Vercel
Backend: Node.js, Express.js, deployed on Railway
Database: MySQL (Aiven Cloud)
Auth: JWT (JSON Web Tokens)

🚀 Features
Role-based login (Admin, Teacher, Student)
Real-time attendance marking
Dashboard for each user role
Secure authentication

⚙️ Setup Locally
git clone https://github.com/nafreenjm2007-maker/smart-attendance-app
cd smart-attendance-app/backend
npm install
npm start
